Responsive Button Test

Icon Buttons (Should maintain size on mobile)

These should remain circular and maintain their size on mobile devices.

Regular Buttons (Should be full-width on mobile)

These should become full-width on mobile devices (≤640px).

Shimmer Buttons (Should be full-width on mobile)

These should become full-width on mobile devices while maintaining shimmer effects.

Mixed Button Groups

Icon buttons should maintain size, regular buttons should stack vertically on mobile.
